### Development 参与开发
```bash
git clone git@github.com:HelipengTony/antony.git
cd antony
yarn install
```
``` bash
// Development Mode
yarn run serve
```
``` bash
// Production Mode
yarn run build
```

<br/>

### Features 依赖与实现
+ Vue-Router 路由
+ Vue-mugen-scroll 无限加载
+ Vue-i18n 国际化
+ Nprogress 加载进度条
+ RemixIcon 图标
+ MarkDown-it-vue MD 支持
+ BootStrap-Vue UI 框架
+ Highlight.js 代码高亮
